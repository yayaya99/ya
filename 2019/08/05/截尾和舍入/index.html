截尾和舍入

Posted on 2019-08-05

在执行窄化转换时，必须注意截尾与舍入问题。例如：如果将一个浮点值转换整形值，Java会如何处理？

运行及结果：
(int)above: 0
(int)below: 0
(int)fabove: 0
(int)fbelow: 0

运行结果发现，在将float和double转型为整数值时，总是对该数字执行截尾。如果想要得到舍入的结果，就需要使用java.lang.Math中的round()方法。

运行及结果:
Math.round(above): 1
Math.round(below): 0
Math.round(fabove): 1
Math.round(fbelow): 0

由于round()是java.lang的一部分，因此在使用它时不需要额外地导入。
